# St Valentin App

Mini projet réalisé pour m'exercer et monter en compétence en **React** et **CSS**, que je suis actuellement en train d'apprendre.

L'objectif était de pratiquer:
- la structure d'un projet React
- le styling CSS (flexbox, responsive, effets visels)
- le déploiement d'une application front

## Technologies utilisées
- React
- Vite
- CSS
- Font Awesome

## Objectifs du projet
- Comprendre la logique des composants React
- Gérer des données via un fichier externe
- Travailler le responsive (desktop / mobile)
- Déployer une application sur GitHub Pages

## Setup
1. Copier `exemple.data.js`
2. Renommer en `data.js`
3. Remplir avec vos données

## Installation
```bash
npm install
npm run dev
```
